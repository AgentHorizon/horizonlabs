## 🚀 Getting Started

1. **Clone the repository:**
   `git clone https://github.com/YourGitHubUsername/horizonlabs.git`

2. **Navigate to the desired directory:**
   `cd BASH`

3. **Make the script executable:**
   `chmod +x script_name.sh`

4. **Run the script:**
   `sudo ./script_name.sh`

---

## 🛠️ Requirements

Ensure you have the following installed based on the scripts you intend to use:
* **Bash:** `network-manager`, `openssh-server`, `rsync`
* **Python:** `python3`, `pip`
* **NodeJS:** `node`, `npm`
